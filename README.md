# PHPの画像回転例

**処理・操作の流れ**  

１．html上にて画像をアップロード  
２．画像を回転ボタンで90度ずつ回転（※この時点では画面上で回転しただけで実物画像はそのまま）  
３．保存でsubmit（※この時点でアップロードした画像も回転させる）  
  
  
**page-imgrotate.php**  
htmlに表示させるページ（画像アップロードボタンや回転ボタン）  
  
**imgrotate.js**  
画像アップロード時の画面表示、回転ボタン押下時の見た目上の画像回転、画像削除など  
  
**imgrotate.php**  
submit時に走らせるソース（page-imgrotate.phpに読み込ませる）  
サーバー上の画像を回転させる  
  
※headerやcssは省略
